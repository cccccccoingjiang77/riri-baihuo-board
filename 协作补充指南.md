# 多人协作补充指南（零代码 · 面向非技术同学）

> 目标：不同赛道 / 模块由不同同学补充，**填表的人完全不碰代码、不碰 GitHub**，
> 维护同学（葱）用一个命令一键把内容更新到网页。

---

## 一、整体路径（一句话）

```
运营在 Excel 里填自己那一页  →  发回给葱  →  葱跑一条命令  →  网页自动更新
```

| 角色 | 要做的事 | 门槛 |
|------|----------|:--:|
| 各赛道运营 | 打开 Excel，找到自己赛道的标签页，填/改内容，存盘发回 | 会用 Excel 即可 |
| 维护同学（葱） | 收齐 Excel，运行一条同步命令 | 复制粘贴一行命令 |

> **两种填报形态，选一种即可（数据结果完全一样）：**
>
> **形态 A · 一份总表分标签页（人少时用）**
> 一个《创意卖点填报模板.xlsx》，底部按赛道分标签页，各自填各自那页。
>
> **形态 B · 每赛道一份独立小文件（明细多、分头补时用）**
> `填报-分赛道/` 目录下每个赛道一份独立 Excel（如《填报-家纺.xlsx》）。
> 各赛道同学**只拿自己那一份**，文件小、不怕发错、彼此绝不覆盖，
> 补完各自发回，维护同学把它们放进同一个文件夹一次性同步。
>
> **形态 C · Skill 模式（最推荐！赛道同学连 Excel 都不用碰格式）**
> 把 `skills/track-creative-analysis/` 这个 skill 发给各赛道同学。
> 同学只需把**自己赛道的原始信息**（消耗数据 / 跑量素材口播 / 赛道理解）丢给 AI 助手，
> skill 会以【清洁工具】为黄金样例引导 ta 产出，并**自动生成一份格式完全正确的
> 《填报-XX赛道.xlsx》**回给葱——葱照旧丢进 `填报-分赛道/` 跑一条命令即可。
> 好处：赛道同学不用理解表格结构，产出质量还能对标标杆，格式 100% 对得上回收脚本。

---

## 二、运营同学怎么填（发这段给他们即可）

1. 打开发给你的 **《创意卖点填报模板.xlsx》**。
2. 底部有多个标签页，每个赛道一页。**只填你负责的那一页**（如"家纺""收纳"）。
3. 每页从上到下 5 个部分，**白色格子随便改，黄色是标题、米色是说明不用动**：
   - ① 赛道整体指标（创意条数、CTR、完播、CVR、CPM）——填数字
   - ② 主要卖点词（词云用）——一行一个词 + 权重 0~100，不够就插入行
   - ③ **卖点动因·需求背景**（重点）——为什么这个卖点成立：
     时令/换季/场景/情绪等背景 → 由此产生的真实需求 → 能落到创意的支撑词（用顿号、隔开）
   - ④ 常见爆款话术——一行一条，抄跑量素材里的高转化口播
   - ⑤ 跑量关键点——钩子/节点/福利/明星 四个维度怎么打
4. 填完 **存盘**，把 Excel 发回给葱。完事，你不用管网页。

> 注意：**Top 素材、关键帧、视频这些不用你填**，由系统/维护同学处理，你漏填也不会覆盖掉已有内容。

## 三、维护同学（葱）怎么一键更新

收到运营填好的 Excel（放到项目根目录，覆盖同名文件），在 `site/` 目录下执行：

```bash
cd site

# 方式A：只更新本地网页数据（先本地看效果）
python3 tools/sync_creative_from_excel.py

# 方式B：更新 + 自动提交并推送到 GitHub（一步到位，别人立即能看到）
python3 tools/sync_creative_from_excel.py --push
```

> **不管用总表还是分赛道多份小文件，命令都一样**——脚本会自动判断：
> - 若 `填报-分赛道/` 目录里有分表 → 自动合并所有分表（分头补充场景）；
> - 否则读根目录的《创意卖点填报模板.xlsx》总表。
>
> 也可以手动指定来源：
> ```bash
> python3 tools/sync_creative_from_excel.py 填报-分赛道        # 指定文件夹
> python3 tools/sync_creative_from_excel.py 填报-分赛道 --push  # 合并并推送
> ```

脚本会：
- 读 Excel 里每个赛道页填的内容
- **只覆盖填过的字段**，Top 素材 / 关键帧 / 视频等没在表格维护的内容原样保留（不会误删）
- 自动更新"更新日期"
- 加 `--push` 时自动 commit 并推送，网页 https://cccccccoingjiang77.github.io/riri-baihuo-board/ 随之更新

### 如果模板需要重新生成（比如新增了赛道 / 想重新分发）
```bash
cd site
python3 tools/make_creative_template.py    # 生成【一份总表·多标签页】
python3 tools/make_split_templates.py      # 生成【分赛道·每赛道一份独立小文件】到 填报-分赛道/
```

## 三·五、Skill 模式：赛道同学用 AI 直接产出（推荐形态 C）

**给赛道同学（发这段 + 把 `skills/track-creative-analysis/` 文件夹发给 ta）：**

1. 在装了 CodeBuddy 的环境里，说一句「我要补充 **XX 赛道**的卖点创意分析」，
   AI 会自动加载这个 skill。
2. AI 会先给你看**【清洁工具】黄金样例**（标杆），让你知道"合格长什么样"。
3. 你把手里的原始信息丢给它，能给多少给多少：
   - 赛道指标（创意条数 / CTR / 完播 / CVR / CPM，有报表就贴）
   - 这个赛道最买账的卖点、最痛的痛点
   - 跑量 Top 素材的口播/字幕（最好带时间轴）
   - 你对赛道的理解（季节/场景怎么影响需求、有无明星达人背书）
4. AI 会按标准把它加工好，并生成一份 **《填报-XX赛道.xlsx》**。
5. 你**只需把这份 Excel 发回给葱**，其余不用管——格式已经 100% 对齐，直接能同步进网页。

**skill 内部（维护同学了解即可）：**
```bash
# 赛道同学把结构化好的数据存成 input.json 后，skill 会自动执行：
python3 skills/track-creative-analysis/scripts/build_track_excel.py input.json 填报-分赛道
# 用黄金样例跑通验证：
python3 skills/track-creative-analysis/scripts/build_track_excel.py \
        skills/track-creative-analysis/scripts/example-清洁工具.json
```
- `references/schema.md`：五大模块字段规范（AI 据此加工）
- `references/golden-example-清洁工具.md`：清洁工具黄金样例（标杆）
- 生成的表 sheet 名 = 赛道名，与回收脚本完全对齐，可直接被 `sync` 合并。

## 四、常见问题

- **想让各赛道同学自己产出、还不用懂表格格式，怎么办？** 用**形态 C（Skill 模式）**：把 `skills/track-creative-analysis/` 发给 ta，ta 对 AI 说清楚赛道原始信息，AI 以清洁工具为标杆帮 ta 产出并生成标准 Excel 回给你。
- **明细太多、要不同赛道同学分头补，怎么最省心？** 形态 B/C 都行：形态 B 发 Excel 小文件人工填；形态 C 让 AI 帮着产出，质量更能对标标杆。
- **两个人同时填会冲突吗？** 不会。分赛道各填各的文件；若非要多人同改一份，用腾讯文档/在线表格在线协同，最后导出为 xlsx。
- **填错了会把网页搞乱吗？** 不会。同步前会保留原数据为基底，只覆盖你填的字段；建议葱先用"方式A"本地看一眼再 `--push`。
- **能加卖点词 / 话术吗？** 能，直接在对应区插入行往下写，行数不限。
